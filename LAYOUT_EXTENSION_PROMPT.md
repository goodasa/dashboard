# 현황판 확장 프롬프트

아래 프롬프트를 그대로 복사해서 사용하면, 지금 만든 `dashboard` 폴더 구조와 레이아웃을 유지한 채 기능을 추가하기 쉽습니다.

```text
/dashboard 폴더 안의 현황판을 확장해줘.

반드시 지켜야 할 기준:
1. 현재 다크 테마, 카드 톤, 여백, 타이포 밀도를 유지할 것
2. 읽기 전용 현황판이라는 성격을 유지하고 수정 UI는 넣지 말 것
3. 현재 구조인 "상단 헤더 + 좌측 패널 + 우측 메인 보드" 레이아웃을 무너지지 않게 유지할 것
4. CHUWI 13의 가로 화면에서 멀리서도 읽기 쉽게 유지할 것
5. 모바일에서는 세로로 자연스럽게 쌓이도록 반응형을 유지할 것
6. 데이터 갱신은 폴링 없이 Firebase 실시간 구독 방식만 사용할 것
7. 새 데이터가 없으면 레이아웃이 깨지지 않는 placeholder/empty state를 보여줄 것
8. 수정 대상은 가능하면 `/dashboard` 폴더 안 파일만 사용할 것

이번에 추가할 내용:
- [여기에 추가할 기능을 구체적으로 적기]

원하는 결과:
- 필요한 파일만 수정
- `dashboard/index.html` 기준으로 바로 확인 가능
- 레이아웃이 지저분해지지 않도록 카드/섹션 단위로 확장
- 마지막에 확인 방법까지 짧게 정리
```

## 예시 1

```text
이번에 추가할 내용:
- 좌측 패널에 오늘의 날씨 카드 추가
- 상단 헤더에 온도와 날씨 아이콘 표시
- 날씨 데이터가 없으면 "날씨 연결 예정" placeholder 유지
```

## 예시 2

```text
이번에 추가할 내용:
- 좌측 패널의 고정 일정 자리에 학교/학원 스케줄 Firebase 데이터 연결
- 오전/오후로 나눠서 표시
- 현재 시간 기준으로 진행 중인 일정을 강조
```
