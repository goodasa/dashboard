# 도해 생성 가이드

## 권장 비율

- 권장 비율: `4:3`
- 권장 크기 예시: `1200x900`, `1600x1200`

현재 현황판의 StudySet 카드 도해 영역은 `4:3` 비율에 맞춰져 있습니다.
그래서 모든 도해를 같은 비율로 만들면 화면에서 가장 안정적으로 보입니다.

## 권장 저장 위치

- `memory/images/english/...`
- `memory/images/science/...`
- `memory/images/social/...`

예:

- `memory/images/science/evaporation.webp`

## 초간단 프롬프트

그림만 원할 때는 아래처럼 짧게 쓰는 것이 가장 효과적입니다.

```text
중학생 학습용 개념 그림 한 장을 만들어줘.

단어: [단어]
의미: [의미]

조건:
- 비율 4:3
- 텍스트, 글자, 단어, 자막, 라벨 넣지 말 것
- 영어/한글 삽입 금지
- 노트북 화면, 포스터, 인포그래픽 프레임, 말풍선 금지
- 핵심 장면만 크게
- 흰 여백 최소화
- 단순한 교과서형 도해 스타일
```

## 더 안정적인 프롬프트

생성 결과가 자꾸 복잡해지면 아래 버전을 쓰면 됩니다.

```text
중학생이 3초 안에 핵심 개념을 이해할 수 있는 교육용 도해 이미지를 만들어줘.

[과목]: [과목]
[단어]: [단어]
[의미]: [의미]

출력 조건:
- 이미지 비율은 반드시 4:3
- 그림만 보여줄 것
- 이미지 안에 텍스트, 글자, 단어, 번역, 자막, 라벨을 넣지 말 것
- 노트북 화면, 책 페이지, 포스터, UI 프레임, 워터마크를 넣지 말 것
- 복잡한 포스터가 아니라 한 장면 중심의 교과서형 도해
- 핵심 개념이 바로 보이도록 큰 형태와 단순한 구조 사용
- 불필요한 배경 장식, 그림자, 질감, 효과는 제거
- 색은 3~5개 정도만 사용하고 대비는 높게
- 선은 굵고 형태는 분명하게
- 여백이 많지 않게 주요 대상을 화면 안에 크게 배치
- 정면형, 깔끔한 벡터 스타일

목표:
- [단어]의 의미를 한눈에 이해할 수 있어야 함
- 노트북 현황판 카드 안에서 또렷하게 보여야 함
- 그림 아래쪽에 불필요한 빈 공간이 많지 않아야 함
```

## 파일에 넣는 예시

```md
| 단어 | 의미 | 도해 |
| --- | --- | --- |
| 증발 | 액체가 기체로 변하는 현상 | images/science/evaporation.webp |
```
